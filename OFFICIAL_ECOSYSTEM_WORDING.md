# VV Hybrid Universe Official Wording

This file defines the official short and medium wording for public use across GitHub, web pages, and product surfaces.

## Official Short Description

`VV Hybrid Universe is a local-first ecosystem for identity, memory, access and human-centered intelligence.`

## Official Medium Description

`VV Hybrid Universe is a local-first ecosystem that connects vision, invention, system infrastructure and products around identity, memory, access and human-centered intelligence.`

## Official Long Description

`VV Hybrid Universe is a local-first ecosystem for identity, memory, access and human-centered intelligence. Its public structure is simple: the manifesto explains the vision, VV Hi Framework declares the invention layer, VV Nexus acts as the system core, and products such as LEA, VNEX and VV Pulse turn that structure into real user-facing experiences.`

## Four-Layer Public Structure

- vision: `vv-hybrid-univers-manifesto`
- invention: `vv-hi-framework`
- system core: `vv-nexus`
- products: `vv-lea`, `vv-pulse`, `vv-team`, `VNEX`

## Safe Public Language

Preferred wording:

- `local-first ecosystem`
- `public invention record`
- `system core`
- `human-centered intelligence`
- `products built on top`

Avoid inconsistent wording such as:

- `random experiments`
- `shadow AI empire`
- `just a chatbot`
- `backend orchestrator` as the main public description

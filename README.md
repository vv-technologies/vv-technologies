# VV Hybrid Universe

Local-first products for memory, identity, access and human-centered intelligence.

## Core Products

- [LEA](https://github.com/vv-technologies/vv-lea)  
  Local-first digital companion built to grow with the user.

- [VV Nexus](https://github.com/vv-technologies/vv-nexus)  
  The system core behind the VV Hybrid Universe.

- [VV Pulse](https://github.com/vv-technologies/vv-pulse)  
  Field operations and real-world signal layer.

- [VV Team](https://github.com/vv-technologies/vv-team)  
  Internal coordination and intelligence layer.

## Direction

VV Hybrid Universe is built around a simple idea:

- the user's intelligence should live close to the user
- memory should become more personal over time
- products should respect privacy, identity and control
